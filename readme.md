# Kinesis-tools-rust

This project allow to read an AWS Kinesis stream


## Run Locally

Clone the project

```bash
  git clone https://github.com/Keenegan/kinesis-tools-rust
```

Go to the project directory

```bash
  cd kinesis-tools-rust
```

Read stream

```bash
  cargo run -- --stream <your_stream>
```

Print help
```bash
 cargo run -- --help
